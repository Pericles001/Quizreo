# Quizreo
A web application: simple and minimalistic for people who like quiz games and surveys

## Description

Quizreo is aiming to help people to create survey form and quiz games that they are willing to share with anyone.

* Create new resource
* Update resource
* Delete resource
* Share resource
* Take test
